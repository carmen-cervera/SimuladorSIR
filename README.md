# SimuladorSIR
Simulador interactivo del modelo SIR heterogéneo de propagación de fake news en redes sociales. Desarrollado como material complementario del TFG "Desarrollo y validación de un modelo matemático para la propagación de bulos en redes sociales basado en analogías epidemiológicas SIR" (ETSIT-UPM, 2025/2026).
